# freecodecamp Color Theme

freecodecamp color theme contains two basic themes, light and dark.

## Applying theme

### To apply theme:
* Ctrl+Shift+P -> Preferences: Color Theme -> freecodecamp light
* Ctrl+Shift+P -> Preferences: Color Theme -> freecodecamp dark
